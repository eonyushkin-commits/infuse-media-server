# 🎬 TorrServer to Infuse Media Server

Автоматизированный, полностью контейнеризированный медиасервер для потокового воспроизведения торрентов (фильмов и сериалов) в плеере **Infuse** (Apple TV, iOS, Mac) без предварительного скачивания файлов на диск.

## ⚙️ Архитектура и принцип работы

Проект реализует парадигму *Infrastructure as Code* и состоит из трех Docker-контейнеров:
1. **TorrServer** — движок для потокового стриминга торрентов в оперативной памяти.
2. **Python Parser (Daemon)** — изолированный микросервис, который каждые 5 минут опрашивает TorrServer. Он выполняет пакетное пробуждение новых раздач, скачивает их файловую структуру и генерирует текстовые `.strm` файлы. Имена файлов автоматически очищаются от релизного мусора для 100% точного парсинга метаданных базой TMDB.
3. **WebDAV Server** — легковесный контейнер, который отдает сгенерированные `.strm` файлы в плеер Infuse.

При нажатии "Play" в Infuse, плеер читает прямую HTTP-ссылку из `.strm` файла и начинает стримить видео напрямую с движка TorrServer.

## 🚀 Установка и запуск (на VPS)

Вам потребуется только установленный `docker` и `curl`. Никаких системных зависимостей (Python/Cron) на хост-машине не требуется!

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/torrserver-infuse-webdav.git /opt/infuse-media-server
   cd /opt/infuse-media-server
   ```

2. **Запустите интерактивный установщик:**
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
   *Скрипт спросит у вас желаемые порты и пароли, сгенерирует `.env` файл и автоматически запустит все контейнеры.*

## 🍏 Подключение в Infuse

1. Откройте Infuse -> **Настройки (Settings)** -> **Добавить файлы (Add Files)** -> **Другое... (Other...)**
2. Выберите протокол **WebDAV**.
3. Заполните данные, используйте значения, которые вы ввели при запуске `install.sh`:
   * **Адрес:** `IP_вашего_VPS` (без http://)
   * **Пользователь:** `Ваш логин` (указанный при установке, по умолчанию `admin`)
   * **Пароль:** `Ваш пароль` (указанный при скрытом вводе)
4. В разделе **Дополнительно (Advanced)** укажите **Порт:** (указанный при установке, по умолчанию `8080`).
5. Нажмите **Сохранить** и добавьте новую папку в Избранное (⭐). 

Infuse автоматически просканирует файлы, подтянет постеры, описания и сгруппирует сериалы по сезонам. Новые фильмы будут появляться в Infuse с задержкой не более 5 минут после добавления в TorrServer.

## 🛠 Администрирование

Парсер работает как фоновый демон внутри Docker. Чтобы посмотреть логи в реальном времени (например, увидеть, какие торренты сейчас обрабатываются):
```bash
docker logs -f strm-parser
```

Чтобы остановить весь проект:
```bash
docker compose down
```
